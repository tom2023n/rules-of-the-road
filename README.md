# 🇮🇪 爱尔兰路考规则在线问答学习系统 (Irish Road Rules Quiz)

这是一个专为准备爱尔兰驾驶执照考试（包含理论考及路考口试环节）设计的在线学习平台。该项目收集了 RSA (Road Safety Authority) 官方手册中的核心知识点，通过互动问答的形式帮助考生加深记忆。

🚀 **[点击此处在线体验](https://tom2023n.github.io/rules-of-the-road/)**

---

## 💡 为什么开发这个项目？

在爱尔兰考取驾照，路考（Practical Test）开始前的 **口试环节 (Oral Questions)** 非常关键。考官会随机询问关于交通标志、手势指令以及《Rules of the Road》中的基础知识。

本项目旨在提供一个比阅读 PDF 文档更具互动性、更碎片化的学习方式，方便考生随时随地通过手机或电脑进行复习。

---

## ✨ 功能特点

* ✅ **精选题库**：涵盖了路考中最常被问到的核心交通规则。
* 🔄 **随机练习**：支持题目随机打乱，避免死记硬背顺序。
* 📱 **响应式设计**：完美适配手机、平板及电脑屏幕，方便在等公交或休息时学习。
* ⚡ **即时反馈**：答题后立即显示正确答案，加深印象。
* 🌐 **纯静态部署**：基于 GitHub Pages 部署，加载速度快，无需注册。

---

## 🛠️ 技术栈

这个项目采用了极简且高效的技术方案：

* **HTML5** - 页面结构
* **CSS3** - 样式美化（响应式布局）
* **JavaScript (Vanilla JS)** - 逻辑控制与题目渲染
* **GitHub Pages** - 静态托管

---

## 📖 如何使用

1.  打开 [在线链接](https://tom2023n.github.io/rules-of-the-road/)。
2.  阅读题目并选择/思考你的答案。
3.  点击查看答案进行对照。
4.  通过“下一题”继续练习。
